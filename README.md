# Dataset Parsing Tool

### To Setup

```
git clone https://github.com/AbdulRashidReshamwala/data-parser-caMicroscope.git
cd data-parser-caMicroscope/
pip install -r requirements.txt
```

### To Run

```
python app.py
```
